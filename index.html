営業管理（Kintone） 5項目 +
①リード管理 1項目 +
リード管理とは、見込み顧客（リード）を管理するページです。名刺交換・紹介・問い合わせなどで取得した顧客情報を登録し、案件管理や顧客管理へ連携する営業の入口となります。
①-1 登録方法｜Eight名刺連携 4項目 +
名刺交換の場合はEightから自動登録
自動登録後、Kintoneの「リード管理」を開く
赤枠のファイルボタンをタップ
「顧客管理を新規作成する」へ移行する
①-2 登録方法｜手動登録 4項目 +
名刺交換以外の場合はKintoneから手動入力で登録
右上の「＋」ボタンをタップしてレコードを新規作成
情報入力後に保存
赤枠のプラスボタンをタップし「営業アクション」へ移行
①-3 顧客管理を新規作成する 2項目 +
「顧客管理を新規作成する」をタップし、情報入力後に保存をタップします。
保存すると顧客IDが発行され「案件を登録する」へ連携できます。
①-4 案件管理を登録する 2項目 +
「案件管理を登録する」をタップし、顧客IDの取得から同一の顧客IDを選択し、情報入力後に保存をタップします。
保存をタップすると案件IDが発行され、「案件管理」の「活動を登録する」へ移行します。
②案件管理〜③活動管理、④契約管理、⑤請求管理 1項目 +
営業案件（商談）を管理するページです。顧客管理に紐づく案件を登録し、商談状況・売上見込み・担当者などを管理します。案件は営業活動の中心となり、活動管理・契約管理・請求管理へ連携します。
②-1 案件管理を登録する 4項目 +
リード管理の「案件管理を登録する」をタップし、顧客IDの取得から同一の顧客IDを選択し、情報入力後に保存をタップします。
保存をタップすると案件管理の見込み案件一覧に情報が反映されます。
保存をタップすると案件IDが発行され、「案件管理」の「活動を登録する」へ移行します。
※案件IDが発行されると、「活動管理・契約管理・請求管理」へ連携できます。
②-2 活動を登録する 2項目 +
「活動を登録する」をタップし、案件IDの取得から同一の案件IDを選択します。IDを選択後、情報入力を行い保存をタップします。
保存をタップすると案件管理、顧客管理の顧客管理一覧に情報が反映されます。
②-3 契約書の申請をする 2項目 +
「契約書を申請する」をタップし、顧客IDの取得から同一の顧客IDを選択します。IDを選択後、情報入力を行い保存をタップします。
保存をタップすると案件管理、顧客管理の契約管理一覧に情報が反映されます。
②-4 請求書の申請をする 2項目 +
「請求書を申請する」をタップし、顧客IDの取得から同一の顧客IDを選択します。IDを選択後、情報入力を行い保存をタップします。
保存をタップすると案件管理、顧客管理の請求管理一覧に情報が反映されます。
⑥顧客管理 2項目 +
顧客管理とは、顧客企業および担当者情報を一元管理するページです。
案件・契約・請求などすべてのデータの基盤となり、各管理と紐づけて情報を管理します。
⑦代理店管理 1項目 +
代理店管理とは、代理店企業および担当者情報を一元管理するページです。
⑧ToDo管理 1項目 +
ToDo管理とは、全体のタスク状況を一覧で確認するためのページです。
